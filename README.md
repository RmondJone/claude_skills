# Claude Skills

### springboot-project-scaffold

```bash
npx skills add RmondJone/claude_skills@springboot-project-scaffold -g -y
```

Spring Boot 项目骨架初始化工具，触发词：初始化项目、创建项目骨架、快速启动 Spring Boot 项目。

基于 **Spring Boot 3.2 + MySQL + Redis + RabbitMQ + MyBatis Plus + JWT + Flyway + SpringDoc OpenAPI** 技术栈，一键生成生产级项目结构。
